# Web-Agent Recipe Starter Pack

Use these recipes before starting a browser task. The goal is to make common Google workflows immediately usable while keeping risky actions gated.

## Recipe Status

- `active`: ready to use through a documented command or workflow.
- `draft`: documented and useful, but may need operator judgment.
- `policy-review-required`: do not automate until the operating guidance is reviewed.

## Safety Rule

Stop before any external send, submit, share, publish, purchase, deletion, account change, message posting, or sensitive data entry. Prepare the final screen or content, report it to the user, and continue only after explicit approval in chat.
